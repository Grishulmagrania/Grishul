# Day 2 - Python Operators and Loops
## Objective
To understand advanced Python operators, looping constructs, loop control statements, and solve practice problems using conditional logic and iteration.

---

## Topics Covered

### 1. Bitwise Operators
Studied the working of bitwise operations:
- Bitwise AND (`&`)
- Bitwise OR (`|`)
- Bitwise XOR (`^`)
- Bitwise NOT (`~`)
- Left Shift (`<<`)
- Right Shift (`>>`)

Performed calculations to understand how binary operations work.

---

### 2. Membership Operators
Learned how to check the presence or absence of elements in a collection using:
- `in`
- `not in`

Practiced membership checking with Python lists.

---

### 3. Identity Operators
Explored:
- `is`
- `is not`

Understood the difference between:
- Comparing object identity using `is`
- Comparing object values using `==`

Also observed object memory addresses using `id()`.

---

### 4. Conditional Programming Exercises
Implemented programs to:
- Check whether a number is strictly positive and even.
- Check whether a number is divisible by 3 or 5.

Used logical operators with conditional statements.

---

### 5. For Loop
Learned the syntax and usage of the `for` loop.

Practiced:
- Iterating using `range()`
- Different forms of `range(start, stop, step)`
- Printing number sequences
- Generating multiplication tables

---

### 6. Iterating Through Lists
Used `for` loops to:
- Traverse list elements.
- Print formatted output.
- Calculate the length of strings using `len()`.

---

### 7. Practical Loop Examples
Implemented programs for:
- Password strength checker using string length.
- Counting numbers with different step values.
- Multiplication table generator for user input.

---

### 8. While Loop
Studied the working of the `while` loop.

Implemented:
- Counter-based loop.
- Updating loop control variables to avoid infinite loops.

Activities performed:
- Tested positive, negative, and zero values.
- Verified loop execution for different ranges.
- Checked edge cases for password length.
- Observed output after using different loop control statements.
- Compared expected and actual outputs.

---

### 9. Loop Control Statements
Learned the purpose and implementation of:

#### Break Statement
- Exit a loop immediately when a condition is satisfied.

#### Continue Statement
- Skip the current iteration and continue with the next iteration.

#### Pass Statement
- Use `pass` as a placeholder for future code implementation.

---

### 10. Loop with Else
Understood how the `else` block executes when a loop completes without encountering a `break` statement.

Implemented an example to search for an element in a list.

---

### 11. Practice Programs
Developed programs for:
- Password verification with limited login attempts.
- Age category classification using `if-elif-else`.
- Number-based decision-making using logical operators.
- Loop-based problem solving.
