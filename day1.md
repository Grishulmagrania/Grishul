# Day 1 - Python Fundamental

## Objective
To understand the fundamentals of Python programming, including variables, data types, operators, type conversion, user input, and conditional statements.

## Topics Covered

### 1. Variables and Memory
- Declared and initialized variables.
- Understood the difference between variable names and string literals.
- Used formatted strings (f-strings) for output.
- Learned how to check:
  - Data type using `type()`
  - Memory address using `id()`
- Introduction to variable referencing.

### 2. Dynamic Typing
- Learned that Python is dynamically typed.
- Assigned different data types to the same variable and observed how Python handles type changes.

### 3. Python Data Types
Studied the following built-in data types:
- Integer (`int`)
- Float (`float`)
- Complex (`complex`)
- String (`str`)
- Boolean (`bool`)
- None (`NoneType`)

Also explored:
- Real and imaginary parts of complex numbers.
- Single-line and multi-line strings.

### 4. Mutable vs Immutable Objects
- Introduction to mutable and immutable data types.
- Examples:
  - Immutable: `int`, `float`, `str`
  - Mutable: `list`, `dictionary`

### 5. Type Conversion
- Converted integer values into floating-point values using `float()`.
- Verified the data type before and after conversion.

### 6. Arithmetic Operators
Practiced:
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Floor Division (`//`)
- Modulus (`%`)
- Exponent (`**`)

### 7. Conditional Statements
Implemented:
- Even/Odd number checker using `if-else`.
- User input with `input()` and type conversion using `int()`.

### 8. Practical Problems
Solved:
- Percentage calculator using marks of five subjects.
- Largest among three numbers using logical operators and conditional statements.

### 9. Comparison Operators
Practiced:
- `==`
- `!=`
- `>`
- `<`
- `>=`
- `<=`

### 10. Logical Operators
Understood the usage of:
- `and`
- `or`
- `not`

### 11. Assignment Operators
Practiced shorthand assignment operators:
- `+=`
- `*=`
- `//=`

---

## Practical Work
- Wrote multiple Python programs demonstrating variables, data types, and operators.
- Implemented conditional statements for decision making.
- Practiced user input and type conversion.
- Executed and verified outputs for each example.

---

## Outcome
Successfully completed the fundamentals of Python programming and gained hands-on experience by implementing multiple practice programs covering variables, data types, operators, type conversion, user input, and conditional statements.
