# Day 5 - Tuples, Dictionaries and Sets in Python

## Objective

To understand Python's built-in data structures—Tuples, Dictionaries, and Sets—and learn how to organize, manipulate, and process data efficiently using practical programming examples. The session also focused on solving real-world problems using these data structures.

---

# Introduction

Today's training introduced Python's fundamental data structures that are extensively used in Artificial Intelligence and Machine Learning applications. Data structures help in storing, organizing, and retrieving information efficiently.

The session covered immutable data structures like tuples, key-value mapping using dictionaries, and unique collections using sets. Practical examples demonstrated how these structures simplify data management, preprocessing, and problem-solving in Python.
