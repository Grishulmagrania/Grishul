# Day 3 - Python Functions and Built-in Functions

## Objective

To understand the fundamentals of Python functions, explore the importance of modular programming, differentiate between parameters and arguments, understand the use of the `return` statement, and gain hands-on experience with commonly used built-in Python functions.

---

# Introduction

The third day of training focused on one of the most important concepts in Python—**Functions**. Functions allow programmers to divide a large program into smaller, reusable blocks of code, making programs easier to read, maintain, debug, and reuse.

In Artificial Intelligence and Machine Learning, functions play a vital role in organizing code for tasks such as data preprocessing, feature engineering, model training, prediction, evaluation, and visualization. Almost every ML library, including NumPy, Pandas, Scikit-learn, TensorFlow, and PyTorch, relies heavily on functions.

The session emphasized understanding how functions improve code quality and reduce repetition while making programs modular and scalable.

## 2. Built-in (Inbuilt) Functions

Explored Python's built-in functions that are available without importing any external libraries.

- Lists
- Tuples
- Strings
- Dictionaries
- Sets

Practiced calculating the length of different collections and strings.

---

### `sum()`

Learned how to calculate the total of all numerical values in a collection.

Applications practiced:
- Sum of list elements
- Total marks calculation
- Basic numerical operations

---

### `max()`

Implemented examples using:
- Lists of integers

- Numerical datasets

### `min()`

Practiced with multiple numerical examples.

- Positive numbers
